The code in this folder is for generating mouse example regions and literature search
draw_methylation.R: draw line and dot plot for selected regions
methyl_circle_plot_agnositc.R: function for drawing line and dot plot
mouse_examples_pubmed.R: function to generate relavent literature for each motif we found
mouse_human_overlap_example.R: function to search for literature for the motif in mouse that overlaps with human NME only motif and pick examples in text
mouse_motif_gene_annotation.R: function to generate the target gene for each motif picked from Ken
