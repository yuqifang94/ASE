This folder contains all the script that are used to download the data and from databse. The scripts are from PMID:32728240 with minor modifications
get_files_from_DCC_ATAC.py: download ATAC seq data from ENCODE for mouse
get_files_from_DCC_ChIP.py: download ChIP seq data from ENCODE for mouse
get_files_from_DCC_RNA_seq_tsv.py: download RNA sequencing data from ENCODE for mouse
